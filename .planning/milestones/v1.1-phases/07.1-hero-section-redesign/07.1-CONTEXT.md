# Phase 07.1: Hero Section Redesign - Context

**Gathered:** 2026-03-14
**Status:** Ready for planning
**Source:** PRD Express Path (HERO-REDESIGN-SPEC.md)

<domain>
## Phase Boundary

Replace the current HeroSection + PaycheckHook two-section layout with a single premium hero section. Full-bleed home photo with warm gradient overlay and frosted glass panels containing the homeowner identity and weekly paycheck stat. The stat is the visual centerpiece — oversized Playfair Display in gold at ~112px. The goal: screenshot-worthy, "show a neighbor" quality.

</domain>

<decisions>
## Implementation Decisions

### Layout — Desktop (md+)
- Full-bleed home photo background with warm bottom-heavy gradient overlay
- Two frosted glass panels positioned asymmetrically: top-left (logo + name + address), bottom-right (stat)
- Diagonal reading path (upper-left → lower-right) across the photo
- 3px cream strip at bottom edge for seamless transition to page content
- Height: h-[28rem] (448px)

### Layout — Mobile (<md)
- Full-bleed home photo with stronger gradient (more opacity for readability)
- Logo floats top-left corner — small text fallback, no frosted panel
- Single frosted panel at bottom, full-width: name + address (centered), gold divider, stat (centered)
- Height: h-[35rem] (560px)

### Typography
- $597 stat: Playfair Display 700, 112px desktop / 76px mobile, #C9953E gold, text-shadow, letter-spacing -3px/-2px
- "per week": DM Sans 400, 18px/14-15px, rgba(201,149,62,0.55)
- "Your home earned you": DM Sans 400, 13px/11px, rgba(250,247,242,0.6), uppercase, letter-spacing 2.5px
- Homeowner name: Source Serif 4 600, 22px/17px, rgba(250,247,242,0.95)
- Address: DM Sans 400, 13px/11-12px, rgba(250,247,242,0.55)
- Logo text mobile fallback: DM Sans 700, 13px, rgba(250,247,242,0.5), uppercase, letter-spacing 2px
- New font dependency: Playfair Display 700 — ONLY for hero stat

### Gradient Overlay
- Desktop: linear-gradient(180deg) from rgba(44,44,44,0.1) at top to rgba(44,44,44,0.85) at bottom
- Mobile: stronger — from rgba(44,44,44,0.05) at top to rgba(44,44,44,0.95) at bottom
- Photo visible through top ~35-40%, bottom near-solid where text lives

### Frosted Glass Panels
- background: rgba(44, 44, 44, 0.5)
- backdrop-filter: blur(16px) with -webkit- prefix
- border-radius: 12px (10px mobile)
- border: 1px solid rgba(250, 247, 242, 0.08)
- Desktop padding: 20-28px (name panel), 28-40px (stat panel)
- Mobile padding: 24px

### Stat Styling
- color: #C9953E, Playfair Display 700, 112px/76px, line-height 1.05
- letter-spacing: -3px/-2px, text-shadow: 0 2px 20px rgba(201,149,62,0.2)

### Gold Divider (Mobile Only)
- 1px height, linear-gradient(to right, transparent, rgba(201,149,62,0.3), transparent)
- margin: 0 20px 16px

### Logo
- Source: Google Drive brand logos folder
- Copy to: public/logo-white.png (resize to ~200px wide)
- Desktop: inside top-left frosted panel at ~52px height, opacity 0.85
- Mobile: float top-left without panel, small text fallback or small image

### Negative Equity Handling
- When earningsPerWeek <= 0: skip stat panel entirely
- Hero shows only: home photo + gradient + logo/name panel
- Messaging about shifting values comes in sections below hero
- Avoids drawing attention to negative situation at screenshot-worthy moment

### Components Affected
- DELETE: src/components/homeowner/PaycheckHook.tsx — absorbed into new HeroSection
- MODIFY: src/components/homeowner/HeroSection.tsx — full rewrite
- MODIFY: src/components/homeowner/HomeownerPage.tsx — remove PaycheckHook, pass earningsPerWeek to HeroSection
- MODIFY: src/app/layout.tsx — add Playfair Display font import (700 weight only)
- MODIFY: src/app/globals.css — add --font-playfair CSS variable
- CREATE: public/logo-white.png — resized logo for web

### New HeroSection Props Interface
- clientName: string, address: string, city: string, state: string
- homePhotoUrl: string | null, earningsPerWeek: number
- isNegative derived internally: earningsPerWeek <= 0
